### Effective-binder

### Instruction
```sh
$ chmod +x binder.sh
$ ./binder.sh <#1-6> PORT
```

Once launched it will create a bind shell using your method of choice.

```
OPTIONS :
 1) PHP
 2) NETCAT Traditional
 3) NETCAT OpenBsd
 4) Perl
 5) Python
 6) Ruby
```
